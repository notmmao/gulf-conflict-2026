### 实施内容

1. **创建 `content/index.md`**
   - 项目简介：使用 Qwen Code 自动跟踪战争进展
   - 导航链接：概况、战报
   - 使用 `<!-- include: 文件名.md -->` 语法动态引用内容

2. **修改 `scripts/md2html.py`**
   - 新增 `inject_dynamic_content()` 函数
   - 支持 `<!-- include: filename.md -->` 语法
   - 仅对 `index.md` 进行动态内容注入

3. **生成 `public/index.html`**
   - 运行 `python scripts/md2html.py` 一键转换
   - 输出位置：`public/index.html`

4. **文件结构**
   ```
   content/
   └── index.md          # 主页源文件（含动态引用）
   
   scripts/
   └── md2html.py        # 增加动态注入功能
   
   public/
   └── index.html        # 生成的主页（内容已动态注入）
   ```

5. **使用方式**
   - 更新 `更新记录.md` 后，重新运行构建即可同步到主页
   - 更新 `战报更新.md` 后，同样自动同步
